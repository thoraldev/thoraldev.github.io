# thoraldev.github.io

Portafolio de **Alejandro Toral** (ThoralDev). Full Stack Developer. Publicado en GitHub Pages.

## Subir tu CV

1. Exporta tu currículum en **PDF**.
2. Nómbralo `cv.pdf`.
3. Colócalo en la **raíz** del proyecto (mismo nivel que `index.html`).

El botón **"Descargar CV"** en la página apunta a `cv.pdf`. Si el archivo no existe, el enlace dará 404 hasta que lo añadas.

## Estructura

```
├── index.html
├── cv.pdf              ← Añade aquí tu currículum
├── css/main.css
├── js/main.js
├── favicon.svg
└── README.md
```

## Publicar en GitHub Pages

1. Sube el repo a GitHub (`thoraldev/thoraldev.github.io`).
2. **Settings → Pages**: Source = “Deploy from a branch”, Branch = `main`, folder = `/ (root)`.
3. URL: `https://thoraldev.github.io`.
